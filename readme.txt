A Village Population Census
======================================================

Quick Installation
1. Import these SQL files to a database:
	sensus-rekap.sql
	sensus-users.sql
2. Extract the 'sensus' folder to the server directory
3. Change the necessary setting in file:
	sensus/app/config/database.php

Requirements:
1. Apache server with mod-rewrite
2. MySQL database server
3. PHP 5
